Topik data:
Data pengeluaran iklan TV dan radio di Belanda

Pada pembahasan kali ini akan menghitung data pengeluaran dari iklan TV dan Radio yang ada di Belanda.

Langkah - langkahnya seperti berikut :

1. Membaca data dari URL

Sebelum membaca data dari URL sebelumnya diharuskan untuk memasukkan beberapa library terlebih dulu. Disini ada 3 library yakni pandas untuk mengatur letak data sehingga mudah dicari secara intuitif, matplotlib untuk melakukan visualisasi data seperti membuat plot grafik untuk satu sumbu atau lebih, dan seaborn untuk memvisualisasikan data yang dibangun diatas matplotlib.

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

Selanjutnya untuk membaca dataset yang dimilik tinggal membuat variabel (disini menggunakan url), untuk menampung link atau nama file dataset yang dimiliki, dan buat variabel lagi (disini menggunakan dataset), sebagai wadah untuk membaca datasetnya dan agar bisa dipanggil beberapa kali.

url = "https://vincentarelbundock.github.io/Rdatasets/csv/AER/DutchAdvert.csv"
dataset = pd.read_csv(url)

Untuk mengetahui type data yang dimiliki oleh dataset menggunakan dataset info seperti code dibawah. Maka akan muncul seluruh informasi yang ada di dataset.

2. Membuat statistik dari data setiap atribut yang bertype numerik

Tipe data numerik merupakan tipe data yang mewakili data atau nilai berjenis bilangan, makanya juga dapat disebut tipe data bilangan. Tipe data numerik dibagi atas 3 jenis yaitu integer, float, dan complex.

2.1 Statistik median data TV

Untuk menghitung median sebenarnya sudah ada library nya sendiri yakni median(), disini akan menghitung median atau nilai tengah dari data TV yang ada. Rumus menghitung median adalah
Rumus Median dengan Jumlah data ganjil :

\[M_{e} = \frac{X (n + 1)}{2}\]

Rumus Median dengan Jumlah Data Genap :

\[M_{e} = \frac{X_\frac{n}{2} + X\left (\frac{n}{2}+1 \right )}{2}\]
dataset['tv'].median()

17514.73437

2.2 Statistik median data Radio

dataset['radio'].median()

3705.541259

2.3 Statistik mean data TV

Mean atau rata - rata dihasilkan dari pembagian antara jumlah nilai keseluruhan dengan banyaknya data yang diolah. Mean disimbolkan dengan xÂ¯ rumusnya adalah

\[mean = \frac{x_{1}+x_{2}+x_{3} ... x_{n}}{n}\]
